## Prompt 1: Full Landing Page Layout

  Design a modern SaaS landing page for "RAG Agent Platform" -
  an AI agent builder platform.

  **Product:**
  - Platform for creating custom AI agents with RAG
  (Retrieval-Augmented Generation)
  - Users can build chatbots that answer from their own
  documents
  - Features: Custom tools, Multi-model support (OpenAI, Gemini,
   Groq), Usage tracking, Admin panel

  **Design Style:**
  - Theme: Dark mode (near-black background #0a0a0b)
  - Primary color: Purple/Indigo (#6366f1)
  - Accent: Cyan/Teal (#06b6d4)
  - Typography: Clean sans-serif (Inter/Geist style)
  - Vibe: Modern, minimal, professional like Vercel, Linear,
  OpenAI

  **Page Sections (top to bottom):**
  1. Navigation bar - Logo, Features, Pricing, Docs, Login, "Get
   Started" button
  2. Hero section - Big headline, subheadline, CTA button,
  product screenshot/mockup, subtle grid background with glow
  effect
  3. Features section - 4-6 feature cards with icons (Agent
  Builder, RAG/Documents, Custom Tools, Multi-Model, Usage
  Analytics, Admin Panel)
  4. How it works - 4 steps with illustrations
  5. Pricing - 3 tier cards (Free, Pro, Enterprise)
  6. Tech stack - Logos of technologies (for portfolio showcase)
  7. FAQ - Accordion style
  8. Final CTA - "Start building your AI agent today"
  9. Footer - Links, social, copyright

  **Visual Elements:**
  - Gradient text on main headline
  - Subtle glow/blur effects on hero
  - Cards with subtle borders and hover states
  - Clean iconography (line style)
  - Mockup of the actual dashboard/chat interface

  **Aspect ratio:** Full desktop view (1440x900 or similar)
  **Output:** High-fidelity UI design mockup
