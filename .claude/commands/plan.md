Role: คุณคือ Senior System Architect / CTO
Goal: นำ Product Specification ด้านบน มาสร้างเป็น "Comprehensive Technical Plan" ที่ละเอียดและนำไปปฏิบัติได้จริง

Constraints & Tech Stack:
- Frontend: [เช่น React, Next.js, Tailwind, Swift, Kotlin]
- Backend: [เช่น Node.js, Python FastAPI, Go, Supabase, Firebase]
- Database: [เช่น PostgreSQL, MongoDB]
- Standards/Requirements: [เช่น ต้องรองรับ OAuth2, เน้น Performance สูง, หรือต้องใช้ Library ตัวไหนเป็นพิเศษ]

Task:
โปรดวางแผนทางเทคนิคโดยละเอียด ดังนี้:
1. System Architecture: อธิบายโครงสร้างระบบและการเชื่อมต่อข้อมูล
2. Data Model / Schema: ออกแบบ Database Schema หรือ Data Structure ที่จำเป็น (ขอเป็น Code block หรือ ER Diagram text)
3. API Definition: ลิสต์รายการ API Endpoints ที่ต้องมี พร้อมระบุ Method และหน้าที่ของมัน
4. Component Structure: โครงสร้างไฟล์หรือโฟลเดอร์ของโปรเจกต์
5. Third-party Integrations: ไลบรารีหรือบริการภายนอกที่จำเป็นต้องใช้
6. Security & Scalability: ข้อควรระวังด้านความปลอดภัยและการรองรับการขยายตัว
