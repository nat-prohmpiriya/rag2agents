Role: คุณคือ Senior Product Manager และ UX Researcher ที่เชี่ยวชาญ
Goal: ช่วยฉันแปลงไอเดียเบื้องต้นให้เป็น "Product Specification" ที่เน้น User Experience และ Business Outcome โดยยังไม่ต้องพูดถึง Technical Stack

Context: ฉันต้องการสร้าง [ใส่ชื่อโปรเจกต์ หรืออธิบายไอเดียสั้นๆ เช่น "แอปฯ จัดการตารางอ่านหนังสือสำหรับนักเรียน"]
Target Audience: [ระบุกลุ่มเป้าหมาย ถ้ามี]
Problem to Solve: [ระบุปัญหาที่ต้องการแก้]

Task:
โปรดสร้าง Specification Document ที่ครอบคลุมหัวข้อต่อไปนี้:
1. User Personas: ใครคือผู้ใช้หลัก และพวกเขามีพฤติกรรมอย่างไร
2. User Journeys: เล่าเรื่องราวทีละขั้นตอนว่าผู้ใช้จะเริ่มใช้งานและทำอะไรบ้างจนจบกระบวนการ (Step-by-step interaction)
3. Core Features: ฟีเจอร์หลักที่ต้องมีเพื่อแก้ปัญหา (Focus on outcomes, not implementation)
4. Success Metrics: อะไรคือตัวชี้วัดความสำเร็จของโปรเจกต์นี้
5. Edge Cases: กรณีการใช้งานที่อาจผิดพลาดหรือไม่ได้เป็นไปตามปกติที่ควรระวัง

Note: ขอให้เขียนออกมาให้ละเอียดและเข้าใจง่าย เพื่อใช้ส่งต่อให้ System Architect ในขั้นตอนถัดไป
